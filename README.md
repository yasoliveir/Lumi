# Lumi
Projeto pessoal - Recomendação de livros através de IA

- Inserção de dados referentes ao gênero literário de preferência;
- Temas que mais gosta de ler;
- Temas que não gosta;
- Apresentação de livros que gosta para refinar a pesquisa feita pela IA;
- Ao final, pretende-se criar um prompt que faça com que a IA indique 5 livros baseado nas informações preenchidas pelo usuário, indicando:
  - nome do livro;
  - autor;
  - ano de lançamento;
  - quantidade de páginas;
  - resumo/prefácio.
